=========================================
Flex 弹性盒子 - 核心概念与属性笔记
=========================================

一、基本概念
├─ Flex 容器（Flex Container）：
│  └─ 设置 display: flex 或 display: inline-flex 的元素
├─ Flex 项目（Flex Item）：
│  └─ 容器的所有直接子元素自动成为 Flex 项目
└─ 开启 Flex 布局后：
   ├─ 项目默认在主轴上排列
   ├─ 项目可以自动伸缩
   └─ float、clear、vertical-align 对项目失效

二、坐标轴系统
2.1 主轴（Main Axis）
├─ 默认方向：水平方向（从左到右）
├─ 起始点：main start（左侧）
├─ 结束点：main end（右侧）
└─ 项目尺寸：main size（宽度）

2.2 交叉轴（Cross Axis）
├─ 默认方向：垂直方向（从上到下）
├─ 起始点：cross start（顶部）
├─ 结束点：cross end（底部）
└─ 项目尺寸：cross size（高度）

三、容器属性（Container Properties）
3.1 flex-direction - 控制主轴方向
├─ row（默认）：主轴水平，从左到右
├─ row-reverse：主轴水平，从右到左
├─ column：主轴垂直，从上到下
└─ column-reverse：主轴垂直，从下到上

3.2 flex-wrap - 控制换行方式
├─ nowrap（默认）：不换行，项目可能溢出
├─ wrap：换行，第一行在上
└─ wrap-reverse：换行，第一行在下

3.3 flex-flow - 简写属性
└─ flex-flow: <flex-direction> <flex-wrap>
   示例：flex-flow: row wrap;

3.4 justify-content - 主轴对齐方式
├─ flex-start（默认）：起始端对齐
├─ flex-end：结束端对齐
├─ center：居中对齐
├─ space-between：两端对齐，间隔相等
├─ space-around：项目两侧间隔相等
└─ space-evenly：所有间隔完全相等

3.5 align-items - 交叉轴对齐（单行）
├─ flex-start：起始端对齐
├─ flex-end：结束端对齐
├─ center：居中对齐
├─ baseline：基线对齐（文字底部对齐）
└─ stretch（默认）：拉伸填充容器

3.6 align-content - 交叉轴对齐（多行）
├─ flex-start：起始端对齐
├─ flex-end：结束端对齐
├─ center：居中对齐
├─ space-between：两端对齐，间隔相等
├─ space-around：项目两侧间隔相等
├─ space-evenly：所有间隔完全相等
└─ stretch（默认）：拉伸填充容器
   注：只有多行时生效（需配合 flex-wrap: wrap）

四、项目属性（Item Properties）
4.1 order - 控制项目排列顺序
├─ 默认值：0
├─ 数值越小越靠前
└─ 可以是负数

4.2 flex-grow - 放大比例
├─ 默认值：0（不放大）
├─ 数值为权重比例
└─ 剩余空间按比例分配

4.3 flex-shrink - 缩小比例
├─ 默认值：1（可缩小）
├─ 数值为权重比例
└─ 空间不足时按比例缩小

4.4 flex-basis - 基准尺寸
├─ 默认值：auto（项目自身大小）
├─ 可以是具体值（如 100px）
└─ 决定项目在主轴上的初始大小

4.5 flex - 简写属性
├─ flex: <flex-grow> <flex-shrink> <flex-basis>
├─ flex: 1; → flex: 1 1 0%;
├─ flex: auto; → flex: 1 1 auto;
└─ flex: none; → flex: 0 0 auto;（固定大小）

4.6 align-self - 单独对齐
├─ 覆盖容器的 align-items 设置
├─ 取值同 align-items
└─ 默认值：auto（继承容器设置）

五、关键术语速查
┌─────────────────┬────────────────────┐
│     术语        │       含义         │
├─────────────────┼────────────────────┤
│ Main Axis      │ 主轴（默认水平）   │
│ Cross Axis     │ 交叉轴（默认垂直） │
│ Main Start/End │ 主轴起点/终点     │
│ Cross Start/End│ 交叉轴起点/终点   │
│ Main Size      │ 主轴方向尺寸      │
│ Cross Size     │ 交叉轴方向尺寸    │
│ Flex Container │ Flex 容器         │
│ Flex Item      │ Flex 项目         │
└─────────────────┴────────────────────┘

六、常用布局模式
├─ 水平居中：justify-content: center
├─ 垂直居中：align-items: center
├─ 两端对齐：justify-content: space-between
├─ 均分空间：justify-content: space-around/space-evenly
├─ 自动换行：flex-wrap: wrap
├─ 等高布局：align-items: stretch（默认）
└─ 项目排序：order 属性

七、注意事项
├─ Flex 容器只会影响直接子元素
├─ align-content 只对多行有效
├─ flex-basis 设置后，width/height 可能被覆盖
├─ 负 margin 仍然有效
└─ 百分比宽度基于容器可用空间

=========================================
总结：Flexbox 是一维布局模型，通过主轴和交叉轴控制项目排列。掌握容器属性和项目属性的配合使用，可以实现各种灵活的布局效果。核心要点：justify-content 控制主轴，align-items/align-content 控制交叉轴，flex 属性控制项目伸缩。
