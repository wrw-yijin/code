HTML基本操作及代码

这是一个用于练习和提交的 HTML 基础示例页面，包含常见标签、属性、列表、表格和表单的基本用法。

1. HTML基本结构

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>页面标题</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>这是一个段落。</p>
</body>
</html>

<p>这是一个普通段落，<strong>这里加粗</strong>，<em>这里斜体</em>。</p>
