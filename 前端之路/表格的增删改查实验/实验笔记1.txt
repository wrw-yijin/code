实验名称：表格的增删改查

1. 页面结构
- 使用 <table>、<thead>、<tbody>、<tr>、<th>、<td> 定义联系人表格。
- 通过 table 的 border、cellspacing、cellpadding 控制表格显示效果。
- 使用 <button onclick="..."> 绑定按钮事件，触发 JS 函数。

2. DOM 获取与定位
- document.getElementById("table") 获取表格元素。
- btn.parentNode.parentNode 从按钮定位到当前所在的 <tr> 行。
- row.cells[index] 获取当前行中指定位置的单元格。
- row.cells[i].querySelector('input')：`querySelector()` 的参数是一个 CSS 选择器字符串，这里传入 `'input'`，表示查找当前单元格中第一个 `<input>` 元素。
  - 作用：返回匹配的第一个 DOM 元素（如果没有匹配则返回 `null`），方便读取 `value`。
  - 常用参数：
    - `'#id'`：按 id 查找元素
    - `'.class'`：按类名查找元素
    - `'tag'`：按标签名查找元素
    - `'parent > child'`：按父子关系查找元素

3. 增加功能
- table.insertRow(length) 在表格末尾插入新行。
- row.insertCell(0/1/2/3/4) 在行中插入单元格。
- 通过 innerHTML 填充单元格内容和操作按钮。

4. 删除功能
- row.parentNode.removeChild(row) 删除当前行，实现“删”。

5. 修改功能
- editRow 函数中使用 prompt() 获取用户输入并修改单元格文本。
- 注释中还展示了将 td 改成 <input> 编辑，再用 saveRow 读取 input.value 保存的方式。

6. 保存功能
- saveRow 中通过 querySelector('input').value 读取当前单元格输入值。
- 修改完成后恢复操作列按钮为 “编辑” 和 “删除”。

7. 调试手段
- console.log("table.js载入成功") 检查脚本是否加载。
- console.log(row) 查看按钮所在的行对象。

8. 样式说明
- table { width: 100%; } 让表格宽度占满父元素。
- th, td { text-align: center; } 使单元格内容居中显示。

9. CSS 表格样式
- border-collapse: collapse; 合并表格相邻单元格的边框，消除单元格之间的缝隙，使表格更紧凑整洁。
- 配合 HTML 属性 border="1" 和 cellspacing="0" 使用，确保不同浏览器中效果一致。

10. 数据提取与批量提交
- tab.rows 获取表格所有行的 HTMLCollection 集合（包括表头），tab.rows[i] 获取第 i 行（索引从0开始）。
- 遍历数据行时从 i=1 开始，跳过表头行（tab.rows[0]）。
- 通过 row.cells[index].textContent.trim() 提取单元格文本内容，构造成数据对象：
  const rowData = {
      name: row.cells[0] ? row.cells[0].textContent.trim() : '',
      age: row.cells[1] ? row.cells[1].textContent.trim() : '',
      sex: row.cells[2] ? row.cells[2].textContent.trim() : '',
      phone: row.cells[3] ? row.cells[3].textContent.trim() : ''
  };

11. 异步请求与 Promise
- 使用 fetch() 发起 HTTP POST 请求，配置项包括：
  - method: 'POST' 指定请求方法
  - headers: { 'Content-Type': 'application/json' } 设置请求体格式
  - body: JSON.stringify(rowObj) 将对象序列化为 JSON 字符串
- data.map(rowObj => {...}) 实现逐行提交：对数组中每个元素单独发送请求
- Promise.all(requests) 等待所有异步请求完成后统一处理结果
- .then(res => res.json()) 在请求成功后解析响应体为 JSON
- .catch(err => {...}) 捕获并处理请求异常

总结：本实验通过 HTML 表格结构和 JavaScript DOM 操作实现了联系人表格的增、删、改、查功能，掌握了节点定位、单元格访问、元素删除、文本修改、异步请求等核心前端技能，特别是通过 Promise 实现批量数据提交的异步处理模式。
